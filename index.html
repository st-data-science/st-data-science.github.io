三つの領域の実績と知見で企業の目標達成を支援します。


データアナリティクス

データを元に現状を正確に分析し、意思決定を行う領域です。 プロジェクト例）広告の品質評価と効果測定・業務プロセス最適化

人工知能 (AI)

機械学習の技術により、機械に自動的に高度な判断を行わせる領域です。 プロジェクト例）文書の自動分類・商品の自動推薦・物体検知

データベース

データを効率的に、安全に集積・加工・転送する技術です。 プロジェクト例）データベースの構築と運用・データパイプラインの開発

Service

コンサルティング

最新のデータサイエンスの知見から、データを利用した最適なサービスを提案させていただきます。

データ分析

データ解析を通して課題を発見・解決し、業務プロセスの最適化を支援します。

データベースやAIに関するソフトウェア開発をハンズオンで支援いたします。

文書 自動分類の新手法の研究開発

August. 2020

文書の自動分類を高速で処理する手法を提案し開発しました。 提案した手法は 既存の手法の速度を遥かに上回り、かつより高い精度の分類を行えます。 SNS上のニュース記事を利用しその高い実用性を示しました。

  • Artificial Intelligence (AI)
  • Big Data
  • Data Analysis

脳波・視線による広告の評価手法の開発

September, 2019

脳波や視線のデータは前提となる情報が少なく解釈が困難です。 また広告の評価は定量的に行われず、しばしば主観的に評価されてしまいます。 本プロジェクトでは、広告の品質を人工知能を用いて自動的に分析する新手法を考案し、開発しました。

  • Artificial Intelligence (AI)
  • Data Analysis
  • Database

交通所要時間マップ可視化アプリ

December, 2019

転勤や引越しの際に目的地から住宅への勤務時間を可視化するアプリを開発しました。 移動時間の計算に最適化アルゴリズムを採用しています。 また住宅の周辺施設を加味して引っ越す街を推薦するシステムを導入しました。

  • Artificial Intelligence (AI)
  • Data Analysis
  • Database

被保険者のリスク算出モデルの作成

Feb, 2017

車両保険会社の顧客の属性からリスクを算出するモデルを作成しました。 保険料はモデルの提示する被保険者のリスクに応じた保険料を請求します。 これにより、より公平でお手頃な価格設定を提供できます。

  • Statistical Modelling
  • Data Analysis

ウイルスの感染地域と、推移の可視化手法の開発

May, 2020

最新の機械学習の手法をウイルスの感染地帯の分析に応用する、確認の範囲で世界初の試みを行いました。 感染の状況がより直感的に表現され、さらに時系列の推移を可視化できます。 これにより感染が急拡大している地域が一目でわかります。

  • Database
